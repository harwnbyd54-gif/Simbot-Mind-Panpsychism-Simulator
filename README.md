# 🌌 SimBot-Mind-Panpsychism-Simulator

[![Python Version](https://shields.io)](https://python.org)
[![License](https://shields.io)](https://opensource.org)

An advanced, light-weight bio-inspired multimodal AI cognitive architecture simulation. This project introduces a revolutionary framework where the environment and surrounding obstacles are not just static data points, but **active cognitive agents (Environmental Minds)** that possess emotional states, energy levels, and interact dynamically with the robot's central consciousness stack.

---

## 🧠 Core Architecture Pillars

1. **Panpsychism Framework (The Internet of Minds)**: Every environmental stimulus (Danger, Goals, Humans) is instantiated as an independent cognitive node. These nodes calculate the robot's proximity and dynamically adjust their "emotional state" and mental importance.
2. **Pure Perception (Kinematic Core)**: Connects sensory inputs directly to reactive behaviors (Speed vs. Distance), triggering instant braking without heavy historical log parsing overhead.
3. **Computer Vision Layer**: Simulates a Convolutional Neural Network (CNN) feature extraction process by analyzing raw pixel intensity matrices to identify and classify objects (Humans, Concrete Walls, Energy Cores) and compute visual threat indices.
4. **Dual-Layer Auditory Loop**: Maintains a low-energy "Zen State" (Passive Hearing) until a critical decibel threshold or a specific Natural Language Processing (NLP) keyword triggers full active cognitive processing.
5. **Vectorized Attention Tension Allocation**: A mathematical approach that maps and prioritizes mental focus based on the highest tension vector among spatial TSP navigation, auditory alerts, and visual overrides operating at $O(N)$ efficiency.
6. **Combinatorial Spatial Optimization**: Integrates a global 300-city Traveling Salesperson Problem (TSP) grid solved via a greedy algorithm to map out optimal long-term energy-efficient navigation lines.

## 🚀 Getting Started

### Prerequisites
Ensure you have Python 3.8+ installed along with the required telemetry and numerical data libraries:
```bash
pip install numpy pandas matplotlib
```

### Running the Simulator
To execute the multi-sensory cognitive clock cycles and observe the unified consciousness logs, run the main script:
```bash
python main.py
```
